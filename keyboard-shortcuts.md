### Keyboard shortcuts

#### Commands
Copy
`ctrl + c`

Paste
`ctrl + v`

Cut
`ctrl + x`

Undo
`ctrl + z`

Redo
`ctrl + y`

Find
`ctrl + f`

#### Windows

Lock Screen
`windows + L`

Screenshot
`shift + windows + s`

Clip Board History(needs to be enabled)
`windows + v`

#### Visual Studio
Comment selected lines
`ctrl + k + c`

Uncomment selected lines
`ctrl + k + u`

Collapse all
`ctrl + m + o`

Uncollapse all
`ctrl + m + p`

Remove using statements
`ctrl + r + g`

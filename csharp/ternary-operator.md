### Ternary Operator

Simple English
`is this condition true ? yes : no`

value = boolean ? true set this : else set this;

Example

`int registrationFee = isSpeaker ? 0 : 50;`

```
int registeration;
if (isSpeaker)
{
  registrationFee = 0;
}
else
{
  registrationFee = 50;
}
```

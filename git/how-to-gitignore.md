### gitignore How to

Use a # for comments and then 

```
# Folder to be ignored
foldername

# Ignore a folder in a folder
foldername/foldername

# Ignore
foldername/filename.extension
filename.extension

# Ignore Anytype of file with the extension
*.extension
```

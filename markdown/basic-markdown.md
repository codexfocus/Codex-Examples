### Basic markdown examples

A couple quick references for [markdown](https://guides.github.com/features/mastering-markdown/).

Inline Code Use
``code here``

Links 
`[GitHub](http://github.com)`

Headers 
`# This is an <h1> tag`

Bold 
`**bold**`

Italic 
`*italic*`

Blockquotes
`> We're living the future so`

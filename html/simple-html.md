### Simple html

Comments

``<!-- Comment Here -->``

Space

`&nbsp;`

Link

`<a href="#">Link Here</a>`

In page link

```
<a id="jump"></a>
<a href="#jump">Go to top</a>
```
Email and Phone links
`<a href="mailto:#">Link Name Here</a>`
`<a href="tel:+18005555555">1-800-555-5555</a>`

Lists
```
<ul>
  <li>#</li>
  <li>#</li>
  <li>#</li>
</ul>

<ol type="1">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol> 
```
